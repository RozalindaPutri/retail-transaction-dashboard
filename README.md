# retail-transaction-dashboard

## Project Overview
This project presents an interactive retail transaction dashboard built using Google Looker Studio.The dashboard provides an overview of sales performance, customer behavior, product performance, and payment method distribution to support data-driven business decisions.

## Key Metrics
- Total Revenue: 8.69M
- Total Customers: 4,000
- Total Profit: 2.0M
- Total Discount Amount: 33.83M
- Total Quantity Sold: 13,559

## Dashboard Insights
- Sales trends show fluctuations across months from 2021 to 2022, with several peak periods indicating high transaction volumes.
- Mobile & Tablets dominate product category revenue, followed by Computing and Entertainment.
- Certain products and SKUs contribute significantly to revenue and profit, indicating key best-selling items.
- Payment methods such as COD and digital payment gateways account for the majority of transactions.
- Customer-level analysis highlights high-value customers contributing substantial revenue.
